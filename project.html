RF-Servo Motor Project

This is a little project I randomly decided to make, it involves connecting an Rf scanner to an Arduino to act as the Input. The Arduino then sends an output to a mini Servo motor and tells it to rotae 90 degrees. I set it up this way so as to act as a locking mechanism.

I plan to add a few things in the future one of those things being a screen to show when it is "locking" and "unlocking". I would also like to show named Ids for seperate Rf Ids used on the screen as an almost account style system it could be some fun
